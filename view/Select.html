    # Select Attributes

    参数 说明 类型 可选值 默认值
    v-model 绑定值 string/number - -

    options 选项列表 array - []
    optionsLabelKey 选项名指定的字段 string - label
    optionsValueKey 选项值指定的字段 string - value
    disabled 禁用 boolean - false
